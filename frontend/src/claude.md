# Src — 前端源代码

React 应用的核心源代码目录。

## 子目录

| 目录 | 职责 |
|---|---|
| [api/](api/) | HTTP 客户端封装，与后端 REST API 通信 |
| [assets/](assets/) | 图片、图标等静态资源 |
| [components/](components/) | 可复用 UI 组件，按功能域分组 |
| [hooks/](hooks/) | 自定义 React hooks |
| [pages/](pages/) | 路由级页面组件 |
| [types/](types/) | TypeScript 接口与类型定义 |

## 入口

- `main.tsx` — 应用入口
- `App.tsx` — 根组件，路由配置
- `App.css` — 全局样式
